Young Spirits Color Editor

Customize your website's color scheme for both Light and Firefly modes

How to Use

  1. Adjust colors using the color pickers or enter hex codes directly
  2. Preview changes in real-time for both Light and Firefly modes
  3. Click "Export Configuration" to download your color scheme as JSON
  4. Copy the exported JSON and save it to src/config/colors.ts in your project
  5. Click "Reset to Defaults" to restore original colors
